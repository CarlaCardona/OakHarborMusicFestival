# OakHarborMusicFestival
A redesigned website for the Oak Harbor Music Festival, that focuses on an improved user experience and modern web design principles.

## About
This project is a complete redesign of the Oak Harbor Music Festival website, created as part of my Interactive Design coursework at Valencia College. The site features a clean, user-friendly interface with improved navigation and visual hierarchy.
